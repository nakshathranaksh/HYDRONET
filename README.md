# HYDRONET💧
HydroNet is a smart, self-sustaining hydroponic farming system using Atmospheric Water Generation (AWG), IoT sensors, and solar power. It enables water-independent, off-grid agriculture with automated monitoring and control. This repo hosts the project’s website and system overview.

## Features
- Simple UI for water system overview
- Prototype model for smart agriculture
- Scalable concept for IoT integration
